## Installation

1. Clone repo
   ```sh
   git clone https://github.com/KiwizBot/KiwizBot-API-Client.git
   ```
2. Install dependancies
   ```sh
   npm i
   ```
3. Complete the configuration
4. Start Bot
   ```sh
   node index.js
   ```
